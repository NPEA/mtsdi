Missing Dataset Statistics

Description

Carry out some statistics from the incomplete dataset

Usage

mstats(dataset)

Arguments

dataset dataset with missing for description

Details

This function computes the proportion of missing observations in a given dataset by rows and columns.

Value

A list containing

rows number of missing in each row
columns number of missing in each column
pattern the pattern of the missing values

Author(s)

Washington Leite Junger wjunger@ims.uerj.br and Antonio Ponce de Leon ponce@ims.uerj.br

Examples
